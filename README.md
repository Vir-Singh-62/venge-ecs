# TypeScript Template

This is a TypeScript template with ESLint and TypeScript configuration pre-set up.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

## Scripts

- `npm run build` - Compiles TypeScript to JavaScript in the `dist` folder
- `npm run dev` - Watches for changes and compiles TypeScript
- `npm run start` - Runs the compiled JavaScript code
- `npm run lint` - Lints TypeScript files
- `npm run lint:fix` - Fixes linting issues automatically

## Structure

- `src/` - Source TypeScript files
- `dist/` - Compiled JavaScript files (generated after build)
- `.eslintrc.json` - ESLint configuration
- `tsconfig.json` - TypeScript configuration
